{% if dhcp_mode == "dnsmasq-proxy" %}

dnsmasq proxy mode is active. This page shows the live proxy values and lets you edit them directly as simple fields.

  • Proxy-DHCP / PXE helper only -- the upstream/router DHCP server keeps issuing every ordinary client's lease and options
  • Every option below (router, NTP, domain, PXE/TFTP boot info, custom options) is delivered only through the supplemental ProxyDHCP/PXE exchange, to PXE/network-boot-aware clients. Ordinary clients never see them; their DNS, gateway, NTP, and domain settings come entirely from the upstream server.
  • No lease-time field: dnsmasq never issues a lease of its own in this mode, so a lease-time value would have no effect. Use Kea mode if you need LanCache NG to control lease time.
  • All fields below are plain text (IPs, a filename, a domain name); no JSON, no client-class syntax, no advanced parser tricks
