Neuen Secondary registrieren

Befehl auf dem Secondary-Host ausführen — ersetzt myname durch einen eindeutigen Namen

bash <(curl -fsSL https://raw.githubusercontent.com/wiki-mod/lancache-ng/master/scripts/setup-secondary.sh) \ --primary {{ primary_url }} \ --token {{ registration_token }} \ --name myname \ --proxy-ip {{ primary_url | replace(from="http://", to="") | replace(from=":8080", to="") }}

Das Skript registriert den Secondary, generiert einen NATS-Token und startet den Container automatisch.
